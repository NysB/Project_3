NBA Dashboard

This dashboard allows users to explosure the historical performance of various NBA teams and gain insights into the players associated with each team.

Change the dropdown below to update the plots

Accessing the NBA Stats API

The following endpoints are available to access data:

  • /api/teams

    Return a list of all NBA Teams

  • /api/player_info

    Return a list of player data

  • /api/player_scores

    Return a list of player scores across a number of years
